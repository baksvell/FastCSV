# Changelog

Все значимые изменения в проекте FastCSV будут документироваться в этом файле.

## [0.2.0] - 2024-12-XX

### Добавлено
- **mmap поддержка для очень больших файлов:**
  - `mmap_reader()` - CSV reader с использованием memory-mapped файлов
  - `mmap_DictReader()` - DictReader с mmap поддержкой
  - 1.14-1.22x ускорение для файлов >10k строк
  - Эффективная работа с файлами, которые не помещаются в RAM
  - Поддержка context manager для автоматического закрытия ресурсов
- 9 новых тестов для mmap функциональности
- Примеры использования mmap (`examples/mmap_usage.py`)
- Бенчмарки для сравнения mmap vs обычного чтения

### Производительность
- mmap_reader показывает 1.14-1.22x ускорение для больших файлов (10k-50k строк)
- Минимальное использование памяти благодаря memory-mapped файлам

## [0.1.0] - 2024-12-XX

### Добавлено
- Базовый парсер CSV на C++17 с SIMD оптимизациями
- Python API совместимый с csv модулем:
  - `reader` - чтение CSV файлов
  - `DictReader` - чтение в словари
  - `writer` - запись CSV файлов
  - `DictWriter` - запись из словарей
- Batch processing для эффективной обработки больших файлов
- Поддержка кастомных разделителей и кавычек
- Обработка многострочных полей в кавычках
- Обработка экранированных кавычек
- Поддержка Unicode символов
- **Функции работы с диалектами:**
  - `register_dialect()` - регистрация кастомных диалектов
  - `unregister_dialect()` - удаление диалектов
  - `get_dialect()` - получение диалекта по имени
  - `list_dialects()` - список всех диалектов
  - Стандартные диалекты: `excel`, `excel-tab`, `unix`
- **Sniffer для автоопределения формата:**
  - `Sniffer.sniff()` - определение delimiter, quotechar, lineterminator
  - `Sniffer.has_header()` - определение наличия заголовка
- 46 тестов покрывающих все основные сценарии (включая mmap)
- Бенчмарки производительности

### Производительность
- Для больших файлов (>10k строк): 0.88x-0.96x скорость стандартного модуля
- Batch processing снижает overhead Python↔C++ вызовов
- SIMD оптимизации эффективны для длинных полей

### Технические детали
- C++17 с AVX2/SSE4.2 SIMD инструкциями
- pybind11 для Python биндингов
- CMake и setup.py для сборки
- Кроссплатформенная поддержка (Windows, Linux, macOS)
